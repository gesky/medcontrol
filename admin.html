Configure o Firebase para ativar o painel

Este painel usa Firebase (Auth + Firestore, e opcionalmente Storage). Para ligá-lo:

  1. Crie um projeto em console.firebase.google.com.
  2. Ative Authentication → E-mail/senha e crie um Firestore.
  3. Cole as chaves do app em js/firebase-config.js.
  4. Crie o primeiro usuário admin manualmente (Auth + documento em users com o mesmo UID, campo role: "admin").

Depois disso, atualize esta página.
